# getCOCO

## Prerequisites

* linux environment
* python
* setuptools
* numpy
* cython

```
pip install (setuptools|numpy|cython)
```

## Installation

Install pycocotools locally :

```
make
```
 Install pycocotools to the python site-packages :
 
 ```
 make install
 ```
 
 ## Usage
 
 ```
 python retrieve_classes.py class1 [class2] [...]
 ```
